# enContact — Teste Prático Frontend

Desafio técnico para vaga de Desenvolvedor Frontend na enContact.

## Stack
- React 19 + TypeScript
- TanStack Router + Query
- Tailwind CSS v4
- i18next (PT/EN) • framer-motion • @dnd-kit

## Funcionalidades
- Lista de mensagens com seleção múltipla (checkbox sobre as iniciais)
- Drag-and-drop entre menus (@dnd-kit)
- Busca + atalhos de teclado
- Marcar como lido/não lido + estrela
- Tema claro/escuro + i18n PT/EN

## Como rodar
\`\`\`bash
npm install
npm run dev
\`\`\`
Acesse  http://localhost:8080/ — login: `Admin` / `Admin`
